# BYD Voice Control - тестовое приложение

## Что делает

Тестирует подключение к `CarServiceProvider` от BYD и пытается выполнить команды управления (включение кондиционера и т.д.) через несколько способов.

## Как собрать APK через GitHub (БЕСПЛАТНО, БЕЗ УСТАНОВКИ)

### Шаг 1: Создай аккаунт на github.com (если нет)

### Шаг 2: Создай новый репозиторий
- Кнопка "+" → "New repository"
- Имя: `byd-voice-control`
- Тип: Public
- Нажми "Create repository"

### Шаг 3: Загрузи файлы
- Нажми "uploading an existing file"
- Перетащи ВСЕ файлы из этой папки
- Нажми "Commit changes"

### Шаг 4: Жди сборку
- Перейди во вкладку "Actions"
- Подожди ~3 минуты пока соберётся
- Когда появится зелёная галочка — нажми на запуск
- Внизу найди "Artifacts" → "BydVoiceControl-debug"
- Скачай ZIP, внутри будет APK

### Шаг 5: Установи на машину
```
adb install app-debug.apk
adb shell am start -n com.byd.voicecontrol/.MainActivity
```

## Что тестировать

1. **"Тест соединения с CarService"** — самая важная кнопка! Она проверяет можем ли мы получить Binder через ContentProvider.

2. **Поле ввода + "Отправить команду"** — вводишь текст ("включи кондиционер"), нажимаешь, видишь что отвечает сервер и пробуешь разные методы.

3. **"🎤 Голосом"** — записываешь голосовую команду через Google.
